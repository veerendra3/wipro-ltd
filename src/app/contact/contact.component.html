Angular 8 Pipes

Pipes are used to Transform Data. It is denoted by symbol | . Pipe takes integers, strings, arrays, and date as input separated with |. It transforms the data in the format as required and displays the same in the browser.



UpperCasePipe & LowerCasePipe

{{ title | uppercase }}

{{ title | lowercase }}

{{ 'Harshal' | uppercase }}


Currency Pipe

{{ 75000.55 | currency:"USD" }}

{{ 75000.55 | currency:"INR" }}


Percent Pipe

{{ 00.675476 | percent }}


Date Pipe

{{ todaydate }}

{{ todaydate | date:'d/M/y' }}

{{ todaydate | date:'d/MMM/y' }}

{{ todaydate | date:'shortTime' }}


Decimal Pipe

{{ 454.78787814 | number:'3.4-4' }}

{{ 5.123456789 | number:'3.1-5'}}


JSON Pipe

{{ jsonval | json }}

Slice Pipe

{{ months | slice:2:6 }}
